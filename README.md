
# RNA-seq Analysis Project

## Overview

This project presents a complete RNA-seq analysis workflow, including quality control, read alignment, differential gene expression analysis, and functional enrichment.

Two datasets were analysed:

- Trypanosoma dataset: comparison between Fat and Blood samples  
- Human dataset: comparison between Male and Female samples  

The analysis was performed using standard bioinformatics tools including FastQC, Trim Galore, STAR, HTSeq, and DESeq2.
